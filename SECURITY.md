# Security Policy

## Supported Versions
| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability
**Do not open a public GitHub issue.**

If you discover a security vulnerability in TruthChat, please email us at security@truthchat.in.
1. Include a Proof of Concept (PoC).
2. We will acknowledge within 24 hours.
